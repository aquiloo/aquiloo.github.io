# Personal Homepage - README

This repository contains the source files for my personal homepage hosted on GitHub Pages.  
The homepage is designed to provide an overview of myself and easy navigation to my projects, blogs, and academic papers.

---

## 1. Home Page (首页)

The **Home Page** should include:

- **Profile Picture / Avatar**  
  Display a clear and professional image of yourself.

- **Self-Introduction**  
  A brief paragraph introducing who you are, your background, and your areas of interest.  
  Example:
  > Hi, I am [Your Name], a software developer and researcher passionate about AI, web development, and open-source contributions.

- **Background / Education / Experience**  
  Highlight your education, work experience, or research focus.  
  Can be displayed as a short bullet list or a paragraph.

---

## 2. Navigation / Jump Pages (可跳转页面)

The homepage should provide clear links or buttons to the following pages:

### 2.1 Projects (项目)

- Showcase your completed or ongoing projects.  
- Each project entry should include:
  - Project name
  - Short description
  - Link to GitHub repository or live demo
  - Technologies used

### 2.2 Blog (博客)

- A page for writing or listing your blog posts.  
- Each blog post entry should include:
  - Title
  - Date
  - Short description
  - Link to the full post (can be Markdown or HTML page)

### 2.3 Papers (论文)

- A page listing your academic publications or research papers.  
- Each paper entry should include:
  - Title
  - Authors
  - Venue / Conference / Journal
  - PDF link or DOI
  - Short abstract (optional)

---

## 3. Optional Features

- **Contact Information**  
  Email, LinkedIn, GitHub profile, or other social links.

- **Theme & Layout**  
  Choose a clean and responsive layout suitable for desktop and mobile browsers.

- **Static Site Generator**  
  Optionally use Jekyll, Hugo, or a simple HTML/CSS structure for easy content management.
